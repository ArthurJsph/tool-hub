# Versão 2.1 - Melhorias de Interface e Mobile

## Alterações Visuais
- **Layout Mobile:** Unificação da barra de navegação. O ícone de menu (hambúrguer) e o perfil do usuário agora ocupam a mesma linha no topo (Header único), removendo a barra duplicada e economizando espaço.
- **Título da Aplicação:** O título da aba do navegador foi padronizado para "Tool-Hub".

## Funcionalidades
- **Configurações:** Adição da seção visual "Zona de Perigo" com botão para "Desativar Conta".
- **Navegação:** Correções parciais no redirecionamento da Sidebar.
