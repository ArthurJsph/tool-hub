# Versão 2.3 - Correções de API, Proxy e Editor

## Resumo da Versão
Esta versão foca na estabilidade da aplicação, corrigindo erros críticos de comunicação entre frontend e backend, e melhorando a experiência de desenvolvimento com ferramentas locais.

## Principais Alterações

### 1. Correção de Erros de API (500/403)
- **Proxy Frontend**: Implementado proxy no Next.js para rotear requisições `/api` para o backend Spring Boot. Isso resolve problemas de CORS e cookies HttpOnly (Erro 403).
- **Versionamento de API**: Atualizados controllers do backend para usar prefixo `/api/v1`, alinhando com a expectativa do frontend.

### 2. Correção de Ferramentas (404)
- **Atualização do Banco de Dados**: A tabela `tools` foi atualizada para corresponder exatamente às rotas do frontend.
- **Ferramentas Adicionadas/Corrigidas**:
  - JSON/JWT Parser
  - Validador de JWT
  - Gerador de UUID
  - Base64 Encoder/Decoder
  - E outras ferramentas essenciais.

### 3. Melhoria no Editor de Código (Monaco Editor)
- **Carregamento Local**: O Monaco Editor agora carrega seus recursos (workers, scripts) localmente do servidor frontend, em vez de usar CDN.
- **Privacidade**: Resolve bloqueios de "Tracking Prevention" em navegadores com configurações de privacidade estritas.

## Arquivos Alterados
- `tool-hub-frontend/next.config.js`: Adicionado rewrite e copy-webpack-plugin.
- `tool-hub-frontend/package.json`: Adicionado `monaco-editor` e `copy-webpack-plugin`.
- `tool-hub-frontend/src/lib/monaco.ts`: Nova configuração do loader.
- `data/init.sql`: Lista de ferramentas atualizada.
- `docker-compose.yml`: Atualizada variável `NEXT_PUBLIC_API_URL`.
