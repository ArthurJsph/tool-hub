# Versão 2.2 - Segurança, Sessão e Estabilidade

## Segurança e Autenticação
- **Armazenamento de Sessão:** Migração de `localStorage` para `sessionStorage`. Agora os dados do usuário e tokens são limpos automaticamente ao fechar a aba do navegador, aumentando a segurança em computadores compartilhados.
- **Fluxo de Logout:** Correção crítica no botão de sair. Agora o usuário é redirecionado forçadamente para a rota `/auth` (Login) ao efetuar logout.

## Backend & Infra (Dev)
- **JWT (Ambiente de Teste):** Aumento do tempo de expiração do Token JWT para facilitar o fluxo de desenvolvimento e testes sem quedas constantes de sessão.

## Performance
- **Otimização de Login:** Redução de consultas ao banco de dados durante a autenticação. O filtro de segurança agora valida o token e extrai as permissões diretamente, sem necessidade de buscar o usuário no banco a cada requisição.
- **Frontend:** Otimização no fluxo de login para utilizar os dados retornados na resposta inicial, eliminando requisições redundantes.
